Gold

Inventory #03642

Locality

Eagle's Nest Mine, Placer County, California, USA

Dimensions

5.7 x 8.9 x 2.9cm

Weight

85g

  • Description

    This crystallized gold specimen from the Eagle’s Nest Mine is an amalgamation of numerous dendritic formations that branch out in an arborescent fashion. Ok tone that down a little so it is little more down to earth, I know we used it before but in retrospect…it is a tad too much…It is affectionately known as, “The Camel” and is a wonderful example of how perfect crystallization and negative space create an unexpected and artful display. It was formerly in the renowned Steven Smale collection, and then later the Marc Weill collection and was de-accessed in 2013.

  • Provenance

    Ex - S. Smale Collection
    Ex - M. Weill Collection
